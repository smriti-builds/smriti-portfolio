# Project

Personal portfolio website for Smriti.

## Goal

Create a premium, editorial-style portfolio website that matches the Figma design exactly.

## Tech Stack

- Next.js 15

- TypeScript

- Tailwind CSS

- Framer Motion

## Rules

- Match Figma exactly.

- Do not redesign.

- Do not invent colors, spacing, or typography.

- Build responsive layouts.

- Reuse components.

- Use semantic HTML.

- Use Framer Motion for animations.
